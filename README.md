# YT Curator

## Introduction

An application to create a daily ordered playlist of YouTube videos.

## Requirements

- Python 3.5 or higher
- Packages detailed in the `requirements.txt` file
- Google API credentials detailed [here](https://developers.google.com/youtube/registering_an_application)

## Setup

- Register the app with Google in order to get a Oauth2 secrets file
- Place this file in the main directory as `secrets.json`

## Usage

To run, execute the following command:

```
python3 curator.py
```
